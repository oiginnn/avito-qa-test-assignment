# Avito QA Internship Assignment

## Описание

В рамках задания протестирован API сервиса объявлений.

Реализованы:
- ручные тест-кейсы
- оформление найденных дефектов
- автоматизированные тесты API

---

## Структура проекта

- `TESTCASES.md` — тест-кейсы
- `BUGS.md` — найденные дефекты
- `test_api.py` — автотесты
- `requirements.txt` — зависимости

---

## Используемые технологии

- Python 3
- pytest — запуск тестов
- requests — HTTP-запросы

---

## Установка

Установить зависимости:

```bash
pip3 install -r requirements.txt
```

---

## Запуск тестов

```bash
pytest -v
```

---

## Что покрыто

- Создание объявления
- Получение объявления по id
- Получение объявлений по sellerId
- Получение статистики

---

## Особенности

- тесты независимы (используется random sellerId)
- не используются фиксированные данные
- проверяются статус-коды и тело ответа
- реализованы позитивные и негативные сценарии
